Simple-ZCU102-Example Documentation
=====================================

Reference firmware and software example for the Xilinx ZCU102 evaluation board
(Zynq UltraScale+ MPSoC, FPGA part ``XCZU9EG-FFVB1156-2-E``). Built on the
SLAC ``axi-soc-ultra-plus-core`` platform; shared workflow documentation lives
on the :hub:`platform docs site <index.html>`.
